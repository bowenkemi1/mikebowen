Concepto CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML. Permite aplicar estilos, como colores, fuentes y layouts, a las páginas web de forma sencilla y eficiente.»
